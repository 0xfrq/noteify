Q1.1 — Hexadecimal Conversion

What is the hexadecimal value of decimal number 18?

Correct Answer: 0x12

Q1.2 — Hex Addition

What is the value of 0x8339e833 + 0x20 ?

Correct Answer: 0x8339e853

Q1.3 — Mixed Base Addition

What is the value of 0x550ecdf2 + decimal 16 ?

Correct Answer: 0x550ece02

Q1.4 — Largest Unsigned 32-bit Integer

What is the largest unsigned 32-bit integer? What happens after adding 1?

max = 0xFFFFFFFF
max + 1 = 0x00000000

Q1.5 — Largest Signed 32-bit Integer

What is the largest signed 32-bit integer? What happens after adding 1?

max = 0x7FFFFFFF
max + 1 = 0x80000000

Q1.6 — Two's Complement

If you interpret an n-bit two’s complement number as unsigned, are negative numbers smaller or larger?

Correct Answer: Larger

Q1.7 — Array Size

How many bytes are needed for char[16] ?

Correct Answer: 16 bytes

Q1.8 — Integer Array Size

How many bytes are needed for int[8] ?

Correct Answer: 32 bytes (assuming 4-byte integers)

Q1.9 — Little Endian 32-bit

Represent pointer 0xDEADBEEF in little-endian 32-bit.

EF BE AD DE

Q1.10 — Little Endian 64-bit

Represent pointer 0xDEADBEEF in little-endian 64-bit.

EF BE AD DE 00 00 00 00

Q1.11 — Character Array Memory Layout

Represent the char array "ABCDEFGH" in little-endian memory layout.

A B C D E F G H
(ASCII sequence in memory order)
